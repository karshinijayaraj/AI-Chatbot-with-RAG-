# AI-Chatbot-with-RAG-
🤖 AI Chatbot with RAG using Streamlit

An intelligent AI chatbot built with Streamlit, LangChain, Ollama, and ChromaDB that supports three powerful modes: RAG (PDF Only), General AI, and Hybrid Mode. The chatbot can answer questions from uploaded PDF documents, general AI knowledge, or a combination of both for context-aware and reliable responses.

🚀 Features
📄 RAG (PDF Only) – Answers questions using information retrieved from uploaded PDF documents.
🧠 General AI Mode – Answers using the LLM's built-in knowledge.
🔄 Hybrid Mode – Combines PDF context with AI knowledge for enhanced responses.
⚡ Fast document retrieval with ChromaDB vector database.
📚 Automatic PDF processing and text chunking.
🔍 Semantic search using embeddings.
🎨 Interactive and user-friendly Streamlit interface.
💾 Persistent vector database for efficient retrieval.

📌 Project Workflow
Upload PDF documents.
Extract and split text into chunks.
Generate embeddings for document content.
Store embeddings in ChromaDB.
Retrieve relevant document sections based on user queries.
Generate context-aware answers using an LLM.
Support General AI and Hybrid modes for flexible interaction.
